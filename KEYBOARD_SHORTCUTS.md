**版本**: 1.0.0

---

## 🌐 全域快捷鍵

| 快捷鍵 | 功能 | 說明 |
|--------|------|------|
| `Cmd/Ctrl + K` | 開啟命令面板 | 搜尋功能、執行命令、快速導航 |
| `Shift + /` | 顯示快捷鍵說明 | 同樣開啟命令面板 |
| `Esc` | 關閉彈窗/對話框 | 關閉 Modal、Command Palette 等 |

---

## 🧭 導航快捷鍵

按住第一個鍵,再按第二個鍵 (例如: 先按 `g`,再按 `d`)

| 快捷鍵 | 功能 | 目標頁面 |
|--------|------|---------|
| `g` → `d` | 前往儀表板 | `/dashboard` |
| `g` → `q` | 前往審核佇列 | `/review/queue` |
| `g` → `a` | 前往審核分析 | `/review/analytics` |
| `g` → `h` | 前往審核歷史 | `/review/history` |
| `g` → `g` | 前往黃金樣本 | `/gold-samples` |
| `g` → `p` | 前往產品總覽 | `/products` |
| `g` → `x` | 前往資料品質 | `/data-quality` |

## 🔍 Command Palette 操作

當 Command Palette 開啟時:

| 操作 | 功能 |
|------|------|
| 輸入文字 | 搜尋命令/產品/記錄 |
| `↑` `↓` | 上下選擇命令 |
| `Enter` | 執行選中的命令 |
| `Esc` | 關閉 Palette |

### 搜尋功能 (NEW! v1.1)

Command Palette 現在支援強大的搜尋功能:

**產品搜尋**:
- 產品名稱 (例如: "燕麥奶")
- 產品 ID (例如: "12345")
- 條碼 (例如: "4710085110012")

**OCR 記錄搜尋**:
- OCR 記錄 ID (完整 UUID 或部分)
- 產品 ID (會顯示該產品的所有 OCR 記錄)

**搜尋歷史**:
- 自動記錄最近 10 次搜尋
- 點擊歷史項目快速重新搜尋
- 支援清除搜尋歷史

## 💡 使用技巧

### 1. 快速導航工作流程

```
情境: 我在產品頁面,想快速切換到審核佇列

操作: g → q
結果: 立即跳轉到 /review/queue
```

### 2. 命令面板搜尋

```
情境 1: 不記得某個功能在哪裡
操作: Cmd+K → 輸入 "審核" → 選擇 "前往審核佇列"
結果: 過濾出所有相關命令,選擇後跳轉

情境 2: 搜尋特定產品
操作: Cmd+K → 輸入 "燕麥奶" → 選擇搜尋結果
結果: 直接跳轉到該產品詳情頁

情境 3: 查看產品的 OCR 記錄
操作: Cmd+K → 輸入產品 ID "12345" → 選擇 OCR 記錄
結果: 跳轉到審核佇列並定位到該記錄

情境 4: 使用搜尋歷史
操作: Cmd+K → (不輸入) → 點擊歷史項目
結果: 快速重複上次的搜尋
```

### 3. 批次審核高效流程 (規劃中)

```
1. g → q                    前往審核佇列
2. x (多次)                  選擇多筆記錄
3. 或 a                      全選當前頁
4. 點擊批次審核按鈕           開啟批次 Modal
5. 選擇評分模板              快速填入
6. Cmd+Enter                提交批次審核
```

## 🎯 快捷鍵設計原則

1. **一致性**: 與常見應用程式保持一致
   - `Cmd/Ctrl + K` 已成為 Command Palette 標準
   - `g` 導航源自 Gmail, GitHub 等

2. **可發現性**: 快捷鍵提示隨處可見
   - Command Palette 中顯示快捷鍵
   - 底部提示列顯示可用操作
   - Tooltip 顯示按鈕快捷鍵

3. **無衝突**: 避免與瀏覽器/系統快捷鍵衝突
   - 不使用 `Cmd/Ctrl + R` (重新整理)
   - 不使用 `Cmd/Ctrl + T` (新分頁)
   - 不使用 `Cmd/Ctrl + W` (關閉視窗)

4. **可記憶**: 使用助記符
   - `g,q` = Go to Queue
   - `g,h` = Go to History
   - `g,g` = Go to Gold samples

## 🆘 常見問題

### Q1: 快捷鍵不起作用?
**A**: 檢查以下項目:
1. 確認焦點在頁面上 (點擊頁面任意處)
2. 檢查瀏覽器擴充功能是否衝突
3. 確認不在輸入框內 (部分快捷鍵在輸入時停用)
4. 重新整理頁面

### Q2: 如何查看所有可用快捷鍵?
**A**: 按 `Cmd/Ctrl + K` 開啟 Command Palette,所有命令都會顯示對應的快捷鍵

### Q3: 可以修改快捷鍵嗎?
**A**: 目前版本尚未支援,計劃在未來版本加入自訂功能

### Q4: Mac 和 Windows 快捷鍵有差異嗎?
**A**: 主要差異在修飾鍵:
- Mac: `Cmd` (⌘)
- Windows/Linux: `Ctrl`

其他按鍵保持一致
